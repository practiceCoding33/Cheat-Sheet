HTML & CSS Cheatsheet

HTML

Element Code Description

Doctype

 <!DOCTYPE html>

Informs the browser that the document is an HTML document.

Span

 <span> </span>

Used to target specific text in a paragraph for styling purposes.

Main

 <main> </main>

Used to contain all of the site's main content.

Main

 <section> </section>

Semantic markup used to contain different sections of site.

CSS

Declaration Code Description

Color

 color: "blue"

Changes the color of text.

BG Color

 background-color: "blue"

Changes the color of an element's background.

Margin

 margin: 20px

Adds margin or space around the outside of an element.

Font Family

 font-family: Georgia

Changes the typeface of text. If name has multiple words, use quotation marks.
